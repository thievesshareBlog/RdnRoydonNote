關於我

Hi，我是 Roydon 😎

曾經是個專職的前端工程師，現在正努力拓展技能邊界，朝著「全端+資安」的夢想邁進。

寫代碼的時候，我是個一絲不苟的碼農；debug 的時候，我則是個冷靜的偵探。總之，沒有 bug 能逃得過我的法眼！（大概吧… 😅）


🚀 技術技能

  • 前端框架：Vue、Angular（是的，我都用！）
  • 後端技術：FastAPI（後端小試身手，但其實更想變強 💪）
  • 網頁設計：HTML、CSS、JavaScript（美感這件事…還在修煉中）
  • 版本控制：Git（commit message 有時候很隨性 🙈）
  • 其他工具：Docker（好用，但還在挖掘更多玩法）

📬 聯絡方式

不提供聯絡方式，因為我知道你肯定會透過某種方式找到我 😏


💡 關於我

熱愛技術，熱愛挑戰，寫 code 之餘也熱愛探索新領域。
我喜愛的哲學語：「百才無用自求歡。」
我不會吹噓自己寫的程式有多完美，畢竟寫 code 本來就是個不斷優化的過程。

目前的目標是讓自己在 資安 繼續努力，成為真正的「資安工程師」！
當然，過程中難免會踩坑，但誰說這不是樂趣的一部分呢？ 😉


如果你對技術有熱情，或只是單純想找個聊聊的人，不妨來找我尬聊一波吧！
