Linux 入門 Day03 - 文字編輯器 & 網路管理

發表於 2024-07-25 更新於 2024-07-26 分類於 Linux Disqus：
文章字數： 3.2k 所需閱讀時間 ≈ 3 分鐘

前言

在現代的計算環境中，網路管理對於 Linux 系統的操作和維護至關重要。無論是個人使用的桌面系統還是企業級的伺服器環境，穩定且高效的網路連接都是確保系統正常運行的關鍵。Linux 以其強大的網路功能和靈活的配置選項，在各種網路環境中廣泛應用。

接下來的內容將深入探討 Linux 網路管理的各個方面，並介紹具體的工具和技術。

常見的文字編輯器

  • emacs ( GNU Emacs )
  • nano ( GNU nano )
  • joe ( Joe’s Own Editor )
  • Vim ( Vi IMproved )

Vim 的常用設定

  • 於編輯器中 命令模式 ( command mode ) 設定
  • 於個人家目錄建立 .vimrc 與內容設定
  • 常用的輔助設定

文字檔案內容的比較

  • diff，兩檔案內容比較 ( 行比較 )
  • diff3，三檔案內容比較 ( 行比較 )
    • diff3 doc1.a doc1.b doc1.c
  • vimdiff，使用 vim 對兩檔案內容比較 ( 行比較 )
    • vimdiff doc1.a doc1.b
  • tr，轉換、刪除指定字符
  • dos2unix、unix2dos，換行字元轉換

常用網路工具

  • ping，遠端主機探測、查詢
  • tracepath，網路路由探測、查詢
  • nslookup，網域正解、反解查詢
  • net-tools，ifconfig、route
  • iproute

DNS 組態

  • /etc/resolve.conf
    • 管理服務：systemd-resolve.service

網路管理套件

  • dhclient、dhcpcd
  • NetworkManager
  • netplan
