Linux 入門 Day02 - 基本文件檔案操作

發表於 2024-07-25 更新於 2024-07-26 分類於 Linux Disqus：
文章字數： 2.6k 所需閱讀時間 ≈ 2 分鐘

前言

在 Linux 和 Unix 系統中，掌握 Shell 指令是提高工作效率和系統管理能力的重要技能之一。Shell 提供了一個強大且靈活的命令行界面，允許使用者執行各種操作，包括文件和目錄的管理。對於初學者來說，學會如何使用 Shell 指令來操作文件和目錄是入門的第一步。

讓我們繼續這段探索 Shell 指令的旅程，揭開命令行世界的神秘面紗！

文件內容輸出

  • cat，將內容輸出於標準輸出 ( standard output，螢幕 )
  • tac，如同 cat 但為反向輸出
  • nl，將內容一行一行輸出於螢幕上，並標註行號

文件瀏覽工具

  • more，全螢幕瀏覽
  • less，全螢幕瀏覽

文件內容擷取

  • head，擷取檔案內容前面內容
  • tail，擷取檔案內容後面內容

資料排序與統計

  • sort，資料排序
  • wc，資料統計

打包備份與解包

  • tar，檔案或資料的封裝，可選擇壓縮功能

內容分割與合併

  • split，內容分割；適用於文字檔、執行檔、壓縮檔
  • cat，內容合併；適用於文字檔、執行檔、壓縮檔

輸出轉向

  • > 輸出轉向
  • >> 輸出附加轉向
  • 2> 錯誤輸出轉向

輸入轉向

  • < 輸入轉向
  • << 輸入附加轉向
  • 2>> 錯誤輸出附加轉向

全部轉向

  • &> 輸出與錯誤輸出轉向
  • &>> 輸出與錯誤輸出附加轉向

管線

  • grep，列出符合指定關鍵字的行數
  • uniq，統計已排序資料之獨特性 ( 以行為處理單位，比較鄰近行 )
